# 8.-
Калькулятор стоимомсти цветов 
#Назнаение программы
Программа расчитывает стоимость юукета цветов с учетом:
- Выбор ттипа цветов
- Количестово цветов 
- Тип упаковки
- Дополнительные услуги (доставка, открытка)
- Скидки (при большом количестве цветов или самовывозе).
#Выходные данные
Полььзователь вводит:
1. Тип цветов (1-розы, 2-тюльпаны, 3-хризантемы)
2. Количество цветов (целое число)
3. Тип упаковки (1-лента, 2-бумага, 3-корзина)
4. Доставка (да/нет)
5. Открытка (да/нет)
6. Самовывоз (да/нет)
#Выходные данные
Программа выводит:
- Базовая стоимость букетов
- Стоимость букетов
- Стоимость допоплнительных усслуг
- Сумма скидки
-Итоговая стоимость букета
#Функциональные требования
1. Расчет базовой стоимости: цена цвета*количество
2. Цена:
 - Розы: 150 руб/шт
 - Тюльпаны: 80 руб/шт
 - Хризантемы: 100 руб/шт
3. Стоимость упаковки:
 - Лента: 50 руб
 - Бумага: 100 руб
 - Корзина: 200 руб
4. Дополнительные услуги:
 - Доставка: 300 руб
 - Открытка: 50 руб
5. Скидки:
 - 5% при количестве цветов более 20 шт
 - 5% при самовывозе
 - Скидки суммируются (максимум 10%)
6. Логирование: кажд расчет сохраняеться в файле log.txt с датой и временем
