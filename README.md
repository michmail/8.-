# 8.-
Калькулятор стоимомсти цветов 
назначаение программы: расчитать стимость букета цветов с учетом выбора цвета цветов,
упаковки, доставки и скидок.
Входные данные: вид цветов ( 1- розы,2-тюльпаны,3-хризантемы)
количество цветов
тип упаковки(лента,бумага,корзина)
доставка(да/нет)
открытка(да/нет)
самовывоз(да/нет - влияет на скидку)
выходные данные: 
базовая стоимость
стоимость упаковки
стоимость доп.услуг
скидка 
итоговая сумма
функциональные требования:
расчет стоимости по формуле(цена цветка*количество)+упаковка+доп_услуги-скидка
скидка:5% при количестве цветов 20 или при самовывозе
логиррование каждого расчета в файл с датой и временем
